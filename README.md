# ThorMace Resource Pack

Turns your Mace into Thor's weapons based on enchantment.

| Enchantment | Skin |
|---|---|
| **Breach** | Stormbreaker - steel-blue axe, Bifrost glow, Groot handle |
| **Density** | Mjolnir - dark Norse hammer with 8-frame animated lightning |

## Requirements
- [CIT Resewn](https://modrinth.com/mod/cit-resewn) (Fabric mod)

## Install
1. Download ZIP from Releases
2. Drop into `.minecraft/resourcepacks/`
3. Enable in Options > Resource Packs
4. Breach enchant = Stormbreaker, Density enchant = Mjolnir
